What is Edge Delivery Services?

Edge Delivery Services is a composable set of services that enables a high degree of flexibility in how you author content for your website.

How do blocks work?

Blocks are reusable components that add styling and functionality to content authored in the CMS. Each block exports a decorate() function that transforms the authored HTML into the final rendered output.

What is da.live?

DA.live is the document-based authoring interface for AEM Edge Delivery Services. Authors use familiar table-based syntax to define blocks and their content.
